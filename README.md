Employee CRUD API

A practice CRUD API project for learning API flow and backend basics.

Features

- Create employee
- Get all employees
- Get employee by ID
- Update employee
- Delete employee

Tech Used

- Python
- SQL
- Git
- GitHub

Project Files

- app_1.py → main API routes
- db_connect.py → database connection
- config.py → configuration settings
- schema.sql → table creation script

Run Project

python app_1.py

API Endpoints

- POST /employees
- GET /employees
- GET /employees/<id>
- PUT /employees/<id>
- DELETE /employees/<id>

Purpose

Created for practice and API testing learning.
